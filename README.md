# RepoInspect_UI# 🌐 RepoInspect Landing Page

This repository contains the source code for [repoinspect.com](https://repoinspect.com) — the professional storefront for the **RepoInspect Deterministic AI Security Engine**.

## 🚀 Overview

RepoInspect is a high-precision security engine built to identify vulnerabilities in AI-native applications (Dify, Mem0, LangChain, etc.). This website serves as the central hub for:
*   **Case Studies**: Deep-dive forensic audits of popular AI frameworks.
*   **Professional Services**: Information on requesting private security audits for enterprise repositories.
*   **Documentation**: High-level architectural overviews.

## 🛠️ Technology Stack

To ensure maximum performance and zero dependencies, this landing page is built with:
*   **Vanilla HTML5 & CSS3**: No frameworks, no bloat, instant load times.
*   **GitHub Pages**: High-availability static hosting.
*   **Google Fonts**: Modern typography (Inter & Outfit).
